# Site Integration Checklist

Use this checklist after adding the library to RoofingHut.

- Add public page at `/resources/`.
- Confirm the page loads `roofinghut_master_library.json` locally.
- Search these phrases and confirm useful results appear:
  - `Goodman GSZ14 2.5 ton`
  - `Rheem hot water tank`
  - `dishwasher not draining`
  - `LG washer dryer`
  - `LiftMaster garage opener`
- Add header/nav link named `Resources`.
- Add footer link named `Manuals & Resources`.
- Add internal links from roofing, solar, HVAC, rebates, products, and contractor pages.
- Mark tech-only resources visibly.
- Do not host copied manufacturer PDF text unless permission allows it.
- Keep outbound manual links pointed at official manufacturer/source pages where possible.
